Définition

Balise input

La balise "input" est utilisé avec plusieurs élements pour créer des champs de textes, des bouttons cliquable, des cases à cocher etc..
Elle doit comporter les attributs suivant : type, name et value.

<input type="button" value="Exemple1">
<input type="button" value="Exemple2">



<input type="radio" name="Exemple" value="Exemple" checked> Exemple1
<input type="radio" name="Exemple" value="Exemple"> Exemple2
