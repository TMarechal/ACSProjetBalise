Définition :

Les balises meta servent à placer des métadonnées (metadata) dans une page HTML. On placera ces informations dans l’élément head, et elles ne seront pas affichées sur la page. Les balises Meta utilisent des valeurs précises, voici quelques éxemples :

  • <meta name="description" content="Une description de la page..." >
    • On parlera de Meta Déscription pour cette ballise, elle affichera une petite description de la page, utile pour le réferencement ou pour avoir un apreçu de la page

  • <meta http-equiv="Content-Type" content="text/html; charset=utf-8>
    • cette balise indiquerait le type de format utilisé, et charset=utf-8 indiquerait le type d'encodage utilisé permettant d'utiliser tous les caractères spéciaux.

D'autres types de meta-données existent mais elles sont un peu moins utilisées :
<meta name="keywords" content="tartiflette, saucisson, maroilles, …" /> cette balise permet d’indiquer un certain nombre de mots-clé à associer à associer à la page,ce sont ces mots clés que les moteurs de recherche utilisent pour éventuellement reférencer la page
<meta name="author" content="RIJARIVO.Afoo" /> <meta name="copyright" content="© Supercalifragilisticexpialidocious Inc." /> Ces balises permettent de fournir un nom d’auteur à une page et une information sur les droits d’auteurs.
