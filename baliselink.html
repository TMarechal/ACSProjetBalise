Définition :

La balise link

La balise link permets de lier une ressource externe à la page HTML. Elle peux servir pour ajouter un fichier de mise en forme, et doit se trouver dans la balise <head>
Exemple :
<head>
<link rel="stylesheet" type="text/css" href="theme.css">
</head>
